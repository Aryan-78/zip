# ZIP Viewer & Editor

A beautiful, lightweight, and modern web-based tool to **view and edit files inside ZIP archives** directly in your browser.

No installation required — just open `index.html` and start working!

![Zip Editor Preview](https://via.placeholder.com/800x400/1a1a2e/7c3aed?text=Zip Editor+Preview)

---

## ✨ Features

- **Modern Dark UI** with smooth animations
- **View Only** and **Editable** modes
- **Image Preview Support** (PNG, JPG, JPEG, GIF, WebP, SVG, BMP)
- **Text File Editing** (HTML, CSS, JS, TXT, JSON, MD, etc.)
- One-click **Save & Download** modified ZIP
- Responsive design (works on desktop & mobile)
- No server needed — runs 100% offline

---

## 🚀 How to Use

### Using the Live Version (Recommended)

1. Go to this link:  
   👉 **[https://aryan-78.github.io/zip/](https://aryan-78.github.io/zip/)**
2. Click on **"Choose ZIP File"** button
3. Select any `.zip` file from your computer
4. Click on any file in the list to view its content:
   - **Images** → Will show preview
   - **Text files** → Can be viewed or edited
5. Use the toggle to switch between **View Only** and **Editable** mode
6. After editing, click **"Save Changes"** to download the updated ZIP file

## ⚠️ Limitations

- **Cannot edit images** inside the ZIP (only preview). You can replace them manually.
- Very large ZIP files may be slow (browser memory limitation)
- Binary files (`.exe`, `.pdf`, `.docx`, etc.) are shown as text — may appear garbled
- Works best with text-based and image files
